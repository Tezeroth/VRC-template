VRChat VCC Template: Master Setup \& Workflow

This repository is a master template for VRChat World development using the VRChat Creator Companion (VCC). It is pre-configured with a Unity-optimized .gitignore to keep the repository clean.



üöÄ How to Start a New Project from this Template

Follow these steps to create a new, independent project without affecting the master template.



1\. Clone the Template

Clone this repository into a new folder named after your project:



Command: git clone https://github.com/Tezeroth/VRC-template.git "Your-New-Project-Name"



Command: cd "Your-New-Project-Name"



2\. Strip Template History

Remove the connection to the template repository so you can start your own version history:



Command: rm -rf .git



Command: git init



Command: git branch -M main



3\. Register with VCC

Open the VRChat Creator Companion.



Go to Projects > Add Existing Project.



Select your new project folder.



Ensure the VCC displays the correct SDK and UdonSharp versions.



4\. Create and Link a New GitHub Repo

Create a brand new, empty repository on GitHub.



Link your local project to that new repository:



Command: git remote add origin https://github.com/Tezeroth/YOUR-NEW-REPO-NAME.git



Command: git add .



Command: git commit -m "Initial commit from master template"



Command: git push -u origin main
